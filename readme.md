# crud-streamers-python

Sistema de gestión de streamers hecho en Python con SQLite. Permite registrar usuarios, iniciar sesión y hacer un CRUD completo de streamers desde la consola.

## Tecnologías usadas

- Python 3
- SQLite3
- Hashlib
- POO

## Funcionalidades

- Registro e inicio de sesión con contraseña encriptada
- Crear, ver, actualizar y eliminar streamers
- Búsqueda por nombre
- Validación de datos
- Protección contra SQL injection


## Estructura

- main.py - menú principal
- controller.py - lógica del CRUD
- models.py - modelos
- authy.py - autenticación
- conexion.py - conexión a la base de datos
